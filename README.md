# 📊 E-Commerce Sales Analytics Dashboard

An interactive, end-to-end data analytics dashboard built with **Python**, **Streamlit**, and **Plotly** to monitor key business metrics and uncover profitability insights for e-commerce sales.

## 🚀 Key Features
- **Real-time KPIs:** Instantly view Total Sales, Profit, Orders, and Average Discount.
- **Monthly Trend Analysis:** Spline charts tracking revenue over time to identify peak sales periods.
- **Category & Sub-Category Insights:** Understand profitability across different product lines and pinpoint the top 5 performing sub-categories.
- **Geographical Distribution:** Interactive pie charts detailing sales by region.
- **Modern UI/UX:** Clean, responsive, and professional interface utilizing custom CSS and the 'Inter' font.

## 🛠️ Tech Stack
- **Python 3.x**
- **Pandas** - Data manipulation, cleaning, and aggregation
- **Streamlit** - Web framework for the interactive dashboard UI
- **Plotly Express** - Dynamic and responsive data visualization

## 📂 Project Structure
- `app.py`: The main Streamlit dashboard application containing the UI and visualization logic.
- `Ecommerce_sales.ipynb`: The Jupyter Notebook containing the initial exploratory data analysis (EDA).
- `Ecommerce_Sales_Data_2024_2025.csv`: The dataset used for the analysis.
- `requirements.txt`: Python dependencies required to run the project.

## ⚙️ How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ibrahim2005507/Ecommerce_sales-Analysis.git
   cd Ecommerce_sales-Analysis
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the dashboard**
   ```bash
   streamlit run app.py
   ```
